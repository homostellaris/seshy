arrow_back Seshy

Welcome to Seshy! There are three core concepts that will help you to understand how it works.

Unsaved sessions

When you first open Seshy you'll see cards representing your browser windows in the unsaved sessions section.

Seshy provides a quick way to switch focus between these windows. Just open Seshy (command + shift + s), navigate to the session you want to switch to (arrow keys), and then select it (enter or left click).

Because these sessions are unsaved they will disappear as soon as their browser window is closed.

The name of each session will be the title of the first tab in the window by default but you can specify a name of your own when you save it.

Unshelved sessions

Saving a session makes it an unshelved session. If the session is not short-lived then you will most likely want to do this.

To save a session click the edit button edit or press r with the session selected. You'll be prompted to give it a name, you can stick with the default or provide one of your own.

It will now appear in the unshelved sessions section (a saved session that is currently open is an 'unshelved' session). If you close the browser window the session will not be lost, it will just move to the shelved sessions section!

Shelved sessions

Shelved sessions can be resumed at any time by selecting their session card (enter or left click). This will re-open the browser window as it was when you closed it.

⚠️ Ensure that you close a saved session by closing the window rather than closing all the tabs. Seshy re-saves sessions after every change to a tab so closing all the tabs results in a saved session with only one tab in it!
